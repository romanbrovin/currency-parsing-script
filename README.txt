1. Скрипт получения ЦБ-курсов валют:
• скрипт получает значение курсов с http://www.cbr.ru/scripts/XML_daily.asp для заданного списка валют;
• для каждой даты и валюты записывает полученные значения в базу данных;
• предполагается, что скрипт запускается с определенной периодичностью и добавляет/обновляет значение курсов валют на каждый день.

2. Виджет, выводящий ЦБ-курсы валют на текущий день:
• виджет показывает ЦБ -курсы валют;
• для каждой валюты должно быть ее текущее значение и изменение по отношению к предыдущему дню (выделять цветом – «красный» - курс упал, «зеленый» - курс вырос);
• так как значение курсов валют может изменяться в течение дня, сделать автоматическое обновление этих значений в виджете с заданным интервалом.

3. Должны быть следующие настройки:
• список валют, курсы которых необходимо получать с ЦБ;
• список валют, которые необходимо выводить в виджете;
• интервал обновления содержимого виджета.


------------


Настройки сервера: PHP 7.4, MySQL 5.7

1. Импортировать дамп (dump.sql) в базу данных
2. В папке config/ открыть файл db.php и внести данные для подключения к базе данных
3. Загрузить все файлы, кроме дампа, в корневую директорию домена
4. Открыть сайт и нажать "Инициализация (сброс)".
   Будут установлены дефолтные параметры.
   Требуется выполнить один раз.
4. Перейти в "Настройки", выбрать необходимые параметры и сохранить
5. Нажать "Обновить курсы" или дождаться автообновления через установленный интервал при открытом сайте


Курсы могу обновляться тремя способами:
1) Нажатием на ссылку "Обновить курсы"
2) Автоматически при открытом сайте
3) Кроном по ссылке http://site.ru/settings/parse-xml

Обновление курсов не произойдет раньше установленного интервала.
К примеру, последнее обновление было сегодня в 15-00.
Интервал обновления 5 минут. Следующее обновление будет не раньше 15-05.
Это необходимо для предотвращения многочисленных запросов к серверу ЦБ.
